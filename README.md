Vesc to BLE written in Rust.

Similar to [nrf52_vesc](https://github.com/vedderb/nrf52_vesc) or [nrf51_vesc](https://github.com/vedderb/nrf51_vesc), but this project support multiple MCU platforms by using [TrouBLE](https://github.com/embassy-rs/trouble) BLE host stack and [embedded-io-async](https://docs.rs/embedded-io-async/latest/embedded_io_async/) for UART.
